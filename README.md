# 🎮 围棋学习游戏

一个基于 HTML5 + JavaScript + Canvas 的围棋入门游戏。

## 功能特点

- **9×9 棋盘** - 适合初学者入门
- **完整围棋规则** - 气的计算、提子、自杀禁手、劫规则
- **学习辅助** - 合法落子提示、悔棋功能
- **交互友好** - 简洁的界面，一目了然的操作

## 如何运行

直接在浏览器中打开 `index.html` 即可开始游戏。

### 本地运行
```bash
# 使用任意 HTTP 服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 操作说明

| 操作 | 说明 |
|------|------|
| 点击棋盘 | 放置棋子 |
| 悔棋 | 回退一步 |
| 重新开始 | 重置棋盘 |
| 弃权 | 连续两次弃权结束游戏 |

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
